Especificaciones técnicas

Hardware y puertos

  • Switch completamente administrable capa L3 (Layer 3 Full)
  • Licencia Network Essentials incluida
  • 24 puertos Gigabit PoE+ (370 W) 10/100/1000 Mbps
  • 04 puertos uplink 10G SFP+ para fibra óptica
  • Montaje en rack (rackeable 1U)

Rendimiento y capacidad

  • Capacidad de switching: 128 Gbps
  • Capacidad con stacking: 208 Gbps
  • Tasa de reenvío (Forwarding): 95.23 Mpps
  • Ancho de banda para stacking: 80 Gbps
  • Soporta stacking de hasta 08 equipos

Recursos de hardware

  • Capacidad de direcciones MAC: 16,000
  • Memoria DRAM: 2 GB
  • Memoria Flash: 4 GB

VLAN y tramas

  • Soporte para hasta 4096 VLAN IDs
  • Jumbo Frames: 9198 bytes

Funcionalidades de capa 3 y servicios

  • Soporta ACLs, DHCP, EIGRP, OSPF, IGMP v1/v2/v3
  • Soporte completo para IPv4 / IPv6

Enrutamiento y capa 3

  • Soporte para IPv4 e IPv6
  • Protocolos de enrutamiento: OSPF, EIGRP, IS-IS, RIP
  • Soporte para acceso enrutado y redes segmentadas

Gestión y monitoreo

  • SNMP v1/v2c/v3 para monitoreo y gestión remota
  • Acceso de administración por Telnet y SSH
  • Interfaz de gestión vía HTTP / HTTPS
  • Compatibilidad con RMON para estadísticas avanzadas
